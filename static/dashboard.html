Root & Health GLOBAL
Server root and health check endpoints — no auth required.
Kirana AI — Auth & Users
Session-based auth with API key or Bearer token. Roles: admin / store_owner.
Kirana AI — Stores
Store management, inventory snapshots, per-store recommendation views.
Kirana AI — Recommendations
ML-powered recommendations: stockout risk, reorder, fast-moving, profit opportunities.
Kirana AI — AI Agents
Natural language queries and Mistral-powered explanations.
Kirana AI — Pipeline
Refresh ML model results from disk.
POS — Auth
OAuth2 password flow. Copy the returned JWT into the auth bar above.
POS — Catalog
Stores, categories, and products from kirana_oltp schema. Prices and stock included.
POS — Orders & Payments
Create orders (auto-deducts stock), list orders, record payments.
POS — Reports
Daily revenue and order stats per store.
WhatsApp — Webhook
Meta webhook verification (GET) and inbound message handler (POST).
WhatsApp — Send Messages
Manually send text, templates, or media. Useful for testing the conversation flow.
WhatsApp — Sessions
Link phone numbers to stores, inspect conversation state, reset sessions.
